## Prerequisites

Create `.env` file with the following content

```
JWT_ACCESS_SECRET=...
JWT_REFRESH_SECRET=...
POSTGRES_DB=...
POSTGRES_USER=...
POSTGRES_PASSWORD=...
```

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
